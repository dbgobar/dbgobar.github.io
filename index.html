Collective Action Organization

HTML | CSS | JQuery | JSON | Third-Party APIs | First-Party API
Ruby on Rails | Heroku

This website was created for a recently funded organization based in
Greece. What makes this site different is that you have the ability to
view a livefeed of texts coming from the refugees living at the refugee
camp-sites. You also have the abiity to donate online and view volunteer
opportunities. I wanted to create something that gave people a real
connection to the refugees in need of our help.

Hey, I'm Bored.

HTML | CSS | JQuery | ReactJS | JSON | Third-Party APIs
First-Party APIs

If you're bored and looking for something to do on the spot, then
use Hey, I'm Bored! An Application that lets you search for events in
your area within a 10 mile radius for that day. You can search from a
list of categories, including; Sports, Family, Arts, Music, Food, and the
Performing Arts.
A fun, quick way to find something to do!
